DEEP RECOMBINANT TRANSFORMER: ENHANCING LOOP COMPATIBILITY IN DIGITAL MUSIC PRODUCTION

Abstract The widespread availability of music loops has revolutionized music production. However, combining loops requires a nuanced understanding of musical compatibility that can be difficult to learn. This study concentrates on the 'vertical problem' of loop compatibility, which pertains to layering different loops to create a harmonious blend. The main limitation to applying deep learning in this domain is the absence of a large, high-quality, labeled dataset containing both positive and negative pairs. We solve this problem by synthesizing high-quality audio from multi-track MIDI datasets containing independent instrument stems and extracting loops as positive pairs. This provides models with instrument-level information when learning compatibility. Previous works create negative samples by altering one of the loops in a positive pair to make it incompatible -- e.g. tempo, key, phase shifts. We argue that altering basic loop characteristics to create negative examples misleads the model to focus on superficial differences over true musical incompatibility. Instead, we use AutoMashUpper for identifying incompatible loops within the same tempo, key, and phase, thus providing more realistic negative pairs for training. Creating a large dataset allows us to introduce and examine the application of Transformer architectures for addressing vertical loop compatibility, achieving an 18.6% higher accuracy than previous state-of-the-art models across multiple genres.
